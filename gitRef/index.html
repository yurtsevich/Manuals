Даведнік па Git

Карысныя спасылкі

Усталёўшчык Git можна спампаваць: http://git-scm.com/download

Афіцыйны навучальны дапаможнік (пачаткоўцам): http://www.kernel.org/pub/software/scm/git/docs/gittutorial.html

Афіцыйны даведнік (адмыслоўцам): http://www.kernel.org/pub/software/scm/git/docs/user-manual.html

Вольная электронная кніга Pro Git: http://progit.org/book

Наладкі

  • ProGit

Наладкі зьмяняюцца пры выкананьні каманды git config і захоўваюцца ў наступных файлах:

  • <ТЭЧКА_РЭПАЗЫТОРЫЮ>/<ТЭЧКА_GIT>/config (звычайна .git/config у корне рэпазыторыю) — наладкі, спэцыфічныя для гэтага рэпазыторыю;
  • <ХАТНЯЯ_ТЭЧКА_КАРЫСТАЛЬНІКА>/.gitconfig (зьмяняюцца пры выкарыстаньні опцыі --global) — наладкі, спэцыфічныя для карыстальніка;
  • <ТЭЧКА_ЎСТАЛЁЎКІ_GIT>/etc/gitconfig (зьмяняюцца пры выкарыстаньні опцыі --system) — наладкі, характэрныя для ўсёй сыстэмы;

Стварэньне рэпазыторыяў

  • ProGit

Існуюць дзьве галоўныя магчымасьці, каб стварыць рэпазыторый. Першая — ініцыіраваць новы рэпазыторый ў лякальнай тэчцы (ў пустой, ці з ужо існуючым праектам). Другая — кланіраваць існуючы "аддалены" рэпазыторый.

Кланаваньне адлеглага рэпазыторыю

"Аддалены" ў дадзеным выпадку дастаткова адноснае разуменьне, таму што такім рэпазыторыям можа быць і існуючы у лякальнай файлавай сыстэме (хаця гэта і ня вельмі верагодны выпадак). Але галоўнае для гэтага мэтаду — што "эталонны" рэпазыторый ужо павінен недзе існаваць:

Магчымыя таксама пратакол http(s):// ці ўвогуле без пратаколу ў выпадку лякальнага зьмяшчэньня "аддаленага" рэпазыторыю. Калі імя ствараемага рэпазыторыю (апошні парамэтар) не пазначана, будзе ўжыта імя "аддаленага" рэпазыторыю.

Падзякі

Пры састаўленьні даведніка выкарыстоўваліся наступныя скрыпты, за якія аўтар выказвае свае падзякі іх аўтарам:

  • jQuery — унівэрсальная JavaScript-бібліятэка
  • XRegExp — JavaScript-бібліятэка для рэгулярных выразаў (regexp)
  • DHTML Tree — скрыпт для пабудовы герархічнага дрэва разьдзелаў даведніка злева.
  • SyntaxHighlighter — скрыпт для каляроўкі фрагмэнтаў коду пэўных моваў/скрыптоў.
